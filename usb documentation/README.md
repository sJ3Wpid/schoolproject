# DigiKeyboard cheatsheet
## Special key translation
The first is the character you use in code, the second charecter is the desired output character
- `IN` = `OUT`
- `"` = `!`
- `!` = `1`
- `)` = `0`
- `/` = `-`
- `-` = `=`
- `*` = `8`
- `@` = `2`
- `&` = `7`
- `#` = `3`
- `{` = `/`
- `}` = `(`
- `]` = `)`
- `1` = `+`
- `>` = `:`
- `(` = `9`
- `_` = `%`
- `:` = `"`
- `^` = `6`
- `\` ` = `;`
